# PROJETO DE INTERFACE — UAITRAMPO

O sistema foi projetado com foco em acessibilidade digital, priorizando usuários com baixa familiaridade com tecnologia, através de:

- Interface simplificada
- Redução de etapas
- Uso de linguagem clara
- Botões grandes e intuitivos

## User Flow

O fluxo de usuário (User Flow) do sistema Uaitrampo foi projetado com foco em simplicidade e facilidade de uso, considerando usuários com pouco conhecimento tecnológico.

O sistema possui dois tipos principais de usuários: candidatos e empresas, e cada um possui um fluxo específico dentro da aplicação.

### Fluxo do Candidato

1- Acessa a tela inicial

2- Clica em “Procurar emprego”

3- Visualiza lista de vagas

4- Clica em uma vaga

5- Visualiza detalhes

6- Clica em “Candidatar-se”

7- Faz login ou cadastro (caso necessário)

8- Confirma candidatura

9- Acompanha em “Minhas candidaturas”

### Fluxo da Empresa

1- Acessa a tela inicial

2- Clica em “Cadastrar vaga”

3- Faz login ou cadastro

4- Acessa painel da empresa

5- Clica em “Nova vaga”

6- Preenche formulário simplificado

7- Publica vaga

8- Visualiza candidatos interessados

## Protótipo

Os protótipos fidelidade do sistema Uaitrampo foram desenvolvidos com o objetivo de validar a estrutura e navegação da aplicação antes da implementação final.

O foco principal foi:

- Simplicidade
- Clareza
- Facilidade de uso

## TELAS PRINCIPAIS

1. Tela Inicial (Home)
- Botão “Procurar emprego”
- Botão “Cadastrar vaga”
- Explicação simples do sistema
- Destaque de vagas

2. Tela de Login / Cadastro
- Campos simples (email e senha)
- Opção:
  Sou candidato
  Sou empresa
  
3. Lista de Vagas
- Cards grandes
- Nome da vaga
- Empresa
- Botão “Ver detalhes”
  
4. Detalhe da Vaga
- Descrição da vaga
- Salário (se houver)
- Botão “Candidatar-se”

5. Minhas Candidaturas
- Lista de vagas aplicadas
Status:
  Enviado
  Em análise
  
6. Dashboard da Empresa
- Botão “Nova vaga”
- Lista de vagas cadastradas
- Acesso aos candidatos

7. Cadastro de Vaga
- Formulário simples:
- Nome da vaga
- Descrição
- Salário
- Botão “Publicar vaga”
